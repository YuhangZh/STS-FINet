# STS-FINet
Pytorch codes of 'Spatial-Temporal Semantic Feature Interaction Network for Semantic Change Detection in Remote Sensing Images' [paper](https://ieeexplore.ieee.org/document/10979855)

# Requirements

python == 3.10.8

torch == 2.1.2+cu121

torchvision == 0.16.2+cu121

einops == 0.8.0

timm == 1.0.3

fonttools == 4.47.0
